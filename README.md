# pc-pg-notes

本仓库用于归档和发布多个独立 HTML 页面，内容涉及各类笔记、工具页面、参考文档等。

这些页面以静态 HTML 形式存放，可直接通过浏览器打开查看，无需构建或服务器环境。

## 目录结构

```
├── pages/          # 存放所有 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/         # 公共资源文件（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，每个文件为一个独立页面。
- `assets/`：存放页面引用的公共资源，如样式表、脚本、图片等。

## 页面归档说明

- 每个 HTML 页面均为独立完整文档，可单独使用。
- 部分页面可能引用 `assets/` 下的资源文件，请确保路径正确。
- 页面内容不针对特定域名或网站，仅为通用笔记或工具页面。

## 使用方式

1. 克隆或下载本仓库。
2. 直接使用浏览器打开 `pages/` 目录下的任意 `.html` 文件。
3. 若需部署，可将整个仓库上传至任何静态托管服务。

## 维护说明

- 新增页面请放入 `pages/` 目录，并保持文件名简洁易识别。
- 如需引用公共资源，请优先使用 `assets/` 中的文件，避免重复。
- 不建议在页面中硬编码外部链接，保持页面自包含性。
- 欢迎提交 Issue 或 Pull Request 改进内容或修复问题。
